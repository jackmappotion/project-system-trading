# Data Sources

## Data Category
- Technical Data
    - Data for technical analysis
    - OHLCV

- Fundamental Data
    - Data for fundamental analysis
    - Netprofit, Assets, Equity, Liabilites etc...

- Info Data
    - Etc 
    - Sector / Product
    - Foreign Trade

### Dart
- sector / product 확인 가능
- 거래중지 및 상장폐지 확인가능

### Finance Datareader
- OHLCV 상당히 정확
- dept 확인가능

### KIS Developer
- OHLCV 상당히 정확
- 매수 / 매도 
- 지정가 예약 가능

### yfinance
- OHLCV 속도 느림
- OHLCV Volume 오류 존재
- 연간 재무제표 속도 좋음

### pykrx
- 외국인 비율 확인가능
- Fundamental factor 바로 확인가능
